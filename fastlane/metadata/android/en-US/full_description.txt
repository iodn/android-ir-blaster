Create custom infrared (IR) remotes using hex codes, raw data, or Flipper Zero .IR files.

Features:

 - Supports Parsed Hex Codes

 - Supports Raw IR signals

 - Multi-transmitter support, send IR via built-in emitter, supported USB IR dongles, or audio-to-IR adapters

 - Signal Tester (bruteforcer) to help discover unknown working IR codes

 - Supports Import of Flipper Zero .IR files

 - Backup / Restore remotes

Note: Requires a phone with a built-in IR emitter to function.
